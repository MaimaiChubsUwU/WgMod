# Notes for sprites
Remember to export at 200% scaling (Terraria does it that way and also helps prevent rotation blurriness in-game)

# UV Armor sprites
UVExtension.pck is an extension Folly made for Pixelorama, made specifically for this project. It adds UV Skin and UV Preview panels.
